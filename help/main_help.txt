**Prefix for this server:** {SERVER_PREFIX}

Documentation for all commands: https://bit.ly/3ALntMg

For help with tabling: `{SERVER_PREFIX}help tabling`
For help with changing MKW TableBot's default settings in your server: `{SERVER_PREFIX}help serverdefaults`
For help with flags on tables: `{SERVER_PREFIX}help flags`
For help with submitting a table in Lounge: `{SERVER_PREFIX}help submittable`
For help with Lounge's Reporter and Updater commands: `{SERVER_PREFIX}help reporters`
For help with anything else: `{SERVER_PREFIX}help other`

*Note:* When you are reading this help documentation, anything inside of parentheses is optional. For example, `{SERVER_PREFIX}setflag (flagcode)` would mean that you do not have to give a **flagcode** when using the command.

If you're new to MKW TableBot and want to learn the basics of tabling, do: `{SERVER_PREFIX}quickstart`
